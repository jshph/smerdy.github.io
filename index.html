About

My name is Joshua. I'm in my junior year studying computer science at UT Austin (B.S. 2018). Here I'm going to share design processes, learnings about software engineering, and other forms of geeking out.

Since 2014 I've been learning web development with the goal of building tools for creativity, productivity, and music. This past summer, I interned on the virtual reality team at Dolby Laboratories focusing on real-time system architecture in C++, which was a huge thrill!

One recent project I'm excited about is Moodify. It spun out of a hackathon project, where I was curious about song categorization by using lyrics instead of audio qualities. My team has documented our pipeline to identify topics in songs using distributional semantics methods, in a Google doc here. Our code is on GitHub here.

I'm nurturing interests in information retrieval, text analysis, and computer vision. This summer, I'll be interning at Spotify working on music recommendation systems!
